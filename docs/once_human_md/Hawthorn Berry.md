Obtaining[]
Hawthorn berries can be gathered from the wilderness, and can be grown by planting Hawthorn Seeds. Hawthorn Berry seeds can be found by random chance when gathering Hawthorn berries from the wilderness, or from the Fetch-A-Lot Bunny, if this deviation is placed inside an Isolated Securement Unit in the player's territory.

Planting[]
Hawthorn Seeds can be planted in Loamy Soil or Planter Boxes, they take 1 hour to mature and need an irrigation level between 50 - 100, and a light level between 50 - 100. For more information on how to cultivate this plant, see the page Seeds and Agriculture.
