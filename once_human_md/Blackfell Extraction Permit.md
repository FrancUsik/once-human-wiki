Blackfell Extraction Permits are lootable items found in storage crates by exploring Strongholds and Settlements. They are used to start up Workshop Defense events.

How To Use[]
Blackfell Extraction Permits are used to start Workshop Defense events at a Workshop. The Workshop does not have to be located in the Blackfell region, for example, you can use a Blackfell Extraction Permit to start a Workshop Event at Refinery Pollution Point (5184,-3608) in the Iron River region.
Each access permit has a tier value which determines which types and level of Monsters will attack, as well as the number of waves the player will encounter during a Workshop Defense event. The tier of the access permit is determined by the level-range of Stronghold, Settlement, or Region the card was acquired from.
