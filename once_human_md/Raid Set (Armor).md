# Raid Set (Armor)

All pieces of the Raid Set armor are of rare rarity. Having 1 of the 6 pieces of the Raid Set equipped increases the players Gathering speed by +15%.
The Raid Set consists of:

Raid Top
Raid Shoes
Raid Pants
Raid Mask
Raid Helmet
Raid Gloves
