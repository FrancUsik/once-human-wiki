Blueberry soda can be unlocked in the Logistics tab of the Memetics page, costing 3ciphers and  200Energy Links, granted the player has unlocked the prerequisite node Air-Drying. It grants the player +50 hydration. The in-game description when crafting any soda will state that it's effect is Glass, with the description: "Foods sealed with Glass will not spoil but such containers can be easily broken." At this moment in time this description is thought to be flavor text, meaning that the soda will not actually break if stored in the player's backpack.
NOTE: As of update 2.1.3, Blueberry Soda is listed under Fruit Soda.
