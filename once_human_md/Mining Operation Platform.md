Summary[]
The Mining Operation Platform is a facility that produces Ore depending on the minerals in the current Region. This process has to be started manually OR can be started automatically with the help of The Digby Boy.

How To Get[]
Building a Mining Operation Platform requires a memetic unlock Infrastructure -&gt; Mining Operation Platform.
